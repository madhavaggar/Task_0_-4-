git clone - is used for making copy of  the repo in a new directory.
cd -is used to change directory to the desktop.
git checkout -b (branch name) - to create a new branch.
touch (file name) - to create a file name.
git merge (branch name) - to merge the specified branchs history into the current one.
git add(file name) - is used to add the files into the local repository and stages them for commit.
git commit -m "message"- is used to save the changes in the local repository.
git push (alias) (branch) - is used to transmit local branch commits to the remote repository branch.
git status - it shows the state of working directory and the staging area.
git mv (file name) (renamed file name) - to rename a file.
